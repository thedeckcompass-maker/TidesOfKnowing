What happens next?

  1. Click PREPARE WEEKLY PACK.
  2. Paste the copied request into Cursor.
  3. Cursor creates the weekly folder and review page.
  4. After Cursor finishes, open START_HERE_FOR_POSTING.html inside the new weekly folder. This file is not at the social-intelligence-system root. It only exists inside the generated week folder.
  5. Review and approve copy, then set the plain Image to use instruction.
  6. Click Export Executive Assistant Posting Handoff.
  7. Optional delivery: after exporting the Executive Assistant handoff, upload the HTML file and any images folder to Google Drive, email, or another shared location.
