# Evergreen Catch-Up Week Campaign Mode

**Applies when:** Source mode is Evergreen Catch-Up Pack or no new article or Field Note is ready.

---

## Purpose

Keep the social schedule active with material that can stand alone, stay useful, and remain accurate without requiring week-specific context.

---

## Core rules

1. **Choose evergreen material** that remains valid over time.
2. **Do not over-explain internal context.** Avoid references to unpublished drafts, internal workflow, or system mechanics.
3. **Prioritise clarity and usefulness** over novelty.
4. **Each post must stand alone.**
5. **Check usage ledger** before selecting a source. Report prior usage factually. Do not block unless Leigh asks to stop.
6. **Exclude two-card pairing pages** unless Leigh confirms they are live.
7. **Exclude hub pages** unless explicitly selected in Campaign Focus / Notes.

---

## Source priority when no URL is supplied

1. Existing Articles
2. Existing Field Notes
3. Strategic Tools
4. Repeating Card Pages
5. Offer Pages only if manually selected in notes

---

## Pack size

Depends on selected source type. Generate the correct fixed pack for the resolved content type.
