# Standard Week Campaign Mode

**Applies when:** No special campaign mode is selected and Campaign Focus / Notes do not indicate first introduction or evergreen catch-up.

---

## Purpose

Standard weekly packs promote one source URL using the default fixed pack structure while still requiring stand-alone, cold-reader-safe posts.

---

## Core rules

1. **Standard weekly packs may use the full 17-asset structure** for articles and Field Notes (14 source-led + 3 training invitation).
2. **Training invitation layer is allowed** on the normal Week A/B/C/D rotation.
3. **Posts must still stand alone.** Even in standard mode, no post should assume the reader has seen other posts from the same week.
4. **Each first line should orient the reader** even when the audience is warmer.
5. **Campaign Focus / Notes, if supplied, are governing inputs.** Apply them to angle selection, CTA stage, and emphasis.

---

## Default pack types by content type

| Content type | Default pack |
| :--- | :--- |
| Article | Full Article Pack, 17 assets |
| Field Note | Full Field Note Pack, 17 assets |
| Tool | Tool Pack, 6 assets |
| Repeating Card Page | Tool Pack, 6 assets |
| Offer Page | Offer Pack, 4 assets |
| About Page | About Introduction Pack (see pack-types/about-introduction-pack.md) |
