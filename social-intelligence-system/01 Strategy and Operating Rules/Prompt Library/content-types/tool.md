# Tool Content Type Prompt

**Applies when:** Source URL is a tool page under `/tools/` or inventory classifies content as Tools.

**Supplements:** `TOK_EDITORIAL_VOICE_AND_GENERATION_GOVERNOR.md`

---

## Purpose

Tool packs drive utility, search discovery, and immediate reader action.

---

## Core rules

1. **Emphasise utility.** Tell the reader what the tool does in plain language.
2. **Name the use case.** When should someone use this tool? What problem does it help with?
3. **Support search discovery.** Pinterest and utility posts should use searchable language.
4. **Invite immediate action.** CTA should be use the tool, explore the tool, or save the prompt.
5. **Do not assume prior knowledge of Tides of Knowing.** Briefly orient new readers if the tool is their first touchpoint.
6. **Assume designed images are needed** unless a strong existing tool screenshot is clearly available.

---

## Pack shape

Default tool pack: 6 assets (see `pack-types/` and generator rules).

---

## What each asset should do

* Explain what the tool is
* Explain when to use it
* Show one practical scenario
* Bridge to search-friendly discovery where relevant
* Invite conversation or reflection on the last Facebook asset
