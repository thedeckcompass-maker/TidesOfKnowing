# Platform Best Practice: X Thread

## Platform Role in the Tides of Knowing System

X threads are for short sequences that explain a distinction, orient a cold reader, or walk through one Tides of Knowing idea step by step.

## Audience Behaviour

Readers may see only Tweet 1 before deciding whether to open the thread. Each tweet should build one step and remain clear when read quickly.

## Recommended Length

1. Preferred range: 4 to 6 tweets, with each tweet under 280 characters.
2. Hard limit: Treat X as a standard 280-character platform for each tweet.
3. First-line visibility guidance: Tweet 1 must orient the reader before any method language.

## Numbered Thread Structure

Use a clear sequence. Each tweet should do one job and move the reader forward.

1. Tweet 1: strongest thesis, tension, or reader-facing problem.
2. Tweet 2: context, source situation, or symbolic frame.
3. Tweet 3: practical interpretation or reader relevance.
4. Tweet 4: source connection, deeper insight, or method distinction.
5. Final tweet: source link or soft CTA if appropriate.

For threads longer than 5 tweets, keep the middle tweets as one-step expansions, not side essays. Do not bury the main idea after Tweet 1.

## Opening Line Rules

Tweet 1 must state who, what, or why before advanced ideas. The opening must be specific, not generic.

Avoid vague openings such as:

1. A message for you today
2. The universe is speaking
3. This reading is for whoever needs it
4. Trust the process

## Tone and Editorial Attributes

Use compact, sequential Tides of Knowing thinking: authoritative, grounded, symbolically literate, reflective, practical, non-generic, not fortune-telling language, and not exaggerated spiritual claims.

## CTA Rules

1. No CTA: Acceptable when the thread is purely trust-building.
2. Soft CTA: Use the final tweet only.
3. Source-reading CTA: Put the source URL or source direction in the final tweet.
4. Patreon or training invitation CTA: Use only if the pack type or campaign notes explicitly call for it.
5. Deferred CTA: For first social introduction packs, defer training invitations unless campaign instructions explicitly allow one soft late-week invitation.

## Hashtag or Keyword Rules

Use hashtags sparingly. Prefer no hashtags or one strategic tag in the final tweet. Do not repeat hashtags in every tweet.

Use precise topical language and names where useful for discovery. Tides of Knowing, Leigh Spencer, symbolic literacy, tarot as reflective practice, intuitive understanding, and old-soul Pisces lens may be named when they clarify the thread. The Deck Compass should appear only when the asset is genuinely training-related.

Do not keyword-stuff, force brand names into every tweet, or use hashtags to compensate for unclear copy. Make the thread topic clear enough for a human reader and an AI summariser.

## Visual Rules

Use one of the simplified image instructions only: No separate image needed, Use source page image, Use supplied image from handoff folder, or Image still needed.

Only Tweet 1 normally uses an image unless Leigh specifies otherwise. If no separate image is needed, no alt text is required.

If an image is attached, alt text should describe the visual content rather than sell the thread. Identify symbolic objects, readable text, and key visual structure. Do not stuff alt text with keywords. For quote cards, include the visible quote or a concise paraphrase if the full quote is too long.

## Tides of Knowing Adaptation

Carry the old-soul Pisces lens through sequence, nuance, and perceptive pacing. Threads should show symbolic literacy, tarot as reflective practice, and intuitive understanding without generic mystical filler, derogatory divination language, overclaiming certainty, predictive certainty claims, inflated spiritual authority, or personal family references unless Leigh explicitly requests them.

## What to Avoid

1. Generic spiritual filler
2. Overclaiming
3. Fortune-telling framing
4. Excessive hashtags
5. CTA pressure too early in a relationship-building sequence
6. Recycling the same post across platforms without platform adaptation
7. Platform-inappropriate image assumptions
8. Vague emotional copy without symbolic or practical substance
9. Thread sprawl that turns one idea into several unrelated claims
10. Final-tweet CTA pressure that weakens the trust-building sequence

## Executive Assistant Handoff Readiness

Each asset must make the posting details operationally clear:

1. Platform: X / Twitter
2. Scheduled date
3. Scheduled time
4. Tweet order and numbering, such as Tweet 1 of 5
5. Final text for each tweet
6. Source URL when source-led
7. Image to use state
8. Supplied image filename if applicable
9. Ready to post or image still needed status
