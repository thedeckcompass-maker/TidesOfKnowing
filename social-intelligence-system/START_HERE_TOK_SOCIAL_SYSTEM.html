Simple weekly workflow

  1. Open the Weekly Pack Builder.
  2. Choose source mode and week start date.
  3. Click PREPARE WEEKLY PACK.
  4. Paste the copied request into Cursor.
  5. Cursor creates the weekly folder.
  6. Open START_HERE_FOR_POSTING.html inside the generated folder.
  7. Review posts, image needs and copy buttons.
  8. Export the Executive Assistant handoff HTML.
  9. Optionally upload the HTML file and any images folder to Google Drive, email, or another shared location.
  10. Send the Drive link to the Executive Assistant.
  11. They post or schedule from START_HERE_FOR_POSTING.html only.

Evergreen Catch-Up: Use Evergreen Catch-Up Pack in the builder when no new article or Field Note is ready. Cursor can select a suitable source from existing inventory.

Posting rules for Executive Assistant

  • Use START_HERE_FOR_POSTING.html only.
  • Use copy buttons.
  • Post or schedule only items marked ready.
  • Do not post needs-image items until the expected files are in the weekly images/ folder.
  • Do not alter wording, links, hashtags or CTAs without Leigh's permission.
  • If uncertain, ask Leigh.

Blocked or excluded

  • Two-card pairing pages are blocked until Leigh confirms they are live.
  • Use only the approved platform set for weekly packs.
  • Hub pages are excluded unless Leigh manually selects them.
  • No automatic posting to social platforms yet.
  • No image generation unless Leigh explicitly requests it.
